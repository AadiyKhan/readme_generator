# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of README Generator
- Automated project scanning and analysis
- AI-powered README generation using Google Gemini
- Technology stack detection
- Dependency detection (Python, Node.js)
- File tree generation
- Entry point detection
- Markdown formatting and cleanup
- Command-line interface with `readme-gen` and `generate-readme` commands
- Support for multiple package managers (pip, npm, poetry, etc.)
- Beautiful terminal UI with progress indicators

### Features
- Scans project structure and analyzes code files
- Detects programming languages and frameworks
- Identifies dependencies from various package manager files
- Generates professional README with badges, features, installation, and usage sections
- Cleans and formats markdown output
- Interactive prompts for project context
